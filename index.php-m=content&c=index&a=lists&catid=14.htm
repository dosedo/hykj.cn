  • Architecting on AWS

           2019年10月底到11月初，翰月科技邀请了Amazon Web Services到公司举办了两期 Architecting on AWS 培训，每期三天。本次培训由认证AWS讲师授课，旨在加深学员对云计算和云最佳实践的认识。学员有机会体验 云政策和云安全方案创建过程，助力学员加速贵机构数字化转型
             Architecting on AWS 介绍了在 AWS 上构建 IT 基础设施的基础知识。课程旨在帮助解决 方案架构师了解 AWS 服务及其如何融入云解决方案，从而教授他们如何优化对 AWS 云的 使用。课程强调了 AWS 云最佳实践和推荐的设计模式，以帮助学员全面考虑在 AWS 上 设计最佳 IT 解决方案架构的过程。课程中的案例研究展示了一些 AWS 客户如何设计自 己的基础设施以及他们所实施的策略和服务。 




      

  • 高效的问题解决力提升

           
          在工作中总是会遇到各种问题，遇到问题并不可怕，可怕的是找不出主要问题，即便找出也不知该如何解决，甚至因错误的解决而产生更大的问题。因此需要对管理人员进行问题解决力的系统提升，通过分析问题的意识、思维培训、分析问题的实用性方法、工具、模型、并最终形成分析与解决问题的可视化结果等系统而量化的内容培训，改变传统的惯性思维与冲动型解决问题的习惯，全面提升中高层管理人员分析与解决问题的能力。
           传统的此类课程侧重于定性的描述策略，此次课程吸收国际先进企业经验，融合本土企业实践，着重从问题分析的思维方式及方法入手，依靠结构化的方法和可视化工具，在此基础上全面解决问题，提升企业绩效，不断改善管理效能。
           此次课程采用线上视频课程教学，学员随时随地利用碎片时间轻松学习。配合线上直播答疑和训后作业强化，采用翻转式学习，帮助学员将课堂效果转化为自身具备的能力，基于公司工作现状的问题分析，将培训效果转化为生产力。
            这是公司首次采用线上培训形式，获得了同事们的一致好评。

  • 逻辑思维与有效表达

            在职场中，做事没逻辑，再累也是白忙活！说话没逻辑，说再多也是听不懂！因为总是无法有逻辑、讲重点，才会出现客户不买账、员工之间相互误解、领导和下属之间相互抱怨的状况。
            无论传达什么信息，都是为了让对方能够更加有效地接收到，有逻辑的沟通则能让沟通的过程更加顺畅，减少不必要的误解和障碍，提升沟通的效率和效果。此外，更加重要的是要具备逻辑思考的能力，能够把复杂、零散的信息结构化、系统化。
           针对以上背景，公司对核心员工进行了为期一天的《逻辑思维与有效表达》培训。培训中，老师对理论部分进行适当的讲解，同事们通过各种模拟情景的演练，使理论与实践相结合，为以后将培训内容应用于实际工作场景做了很好的铺垫。









  • 蝶变：培训师公众表达及授课技巧

          公司的快速发展，离不开企业核心技术、企业文化的不断传承。而企业内部核心骨干在知识传承方面，发挥着非常大的作用。推动企业内部的隐性知识显性化；显性知识成为核心竞争力。因此内训师队伍的建设与培养，就显得非常重要。。这就是我们要搭建内部讲师体系，完善内训师队伍的根本出发点，结合岗位、层级和能力素质模型，将学习嵌入工作中，促进“学习”与“实践”的有效融合。
           培训中，学员们通过大量案例应用、练习，快速、直观、清晰的在模拟培训师成长环境中快速吸收所学知识、技能。大家普遍认为此次培训将对往后工作中开展的各种培训和讲座起到很好的帮助作用。
